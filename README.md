# LOGOTYPE

## To build the site, run the following commands:

```bash
npm install pug
npm install pug-cli -g
pug .\ingex.pug
```

## How to install project 

```bash
git clone https://github.com/vlada-97/blog.git
```

This will generate the site in the same directory as the readme.
